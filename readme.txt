Project 1: Covid Trends in the US
Project Description:
A tool for observing trends in Covid rates among different states in the US

Technologies Used
Apache Spark - version 3.2
Apache Hive - version 2.3.9
Scala Build Tool - version 1.5.5
MySQL - version 8.0.25

Features

Admin/User login system
Looping user interface
Spark queries comunicating with Hive database holding Covid data
MySQL table holding user data
